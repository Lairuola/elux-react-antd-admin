# 项目介绍

本项目主要基于Elux+Antd构建，包含React版本和Vue版本，旨在提供给大家一个简单、开箱即用的后台管理系统通用模版，主要包含运行环境、脚手架、代码风格、基本Layout、状态管理、增删改查逻辑、列表、表单等。

为保持工程简单清爽，方便二次开发，不提供各种纷杂的具体业务组件（网上很多）
